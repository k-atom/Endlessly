## 為了什麼
Endlessly的出生是為了讓人民持續消費，提升人民的消費意願，提高市場的金流。
> 富有來自於金流量，存錢不會使人真正的富有。

## 如何解決
* 貨幣無法無償的囤積

		系統結算時自動收回30日內未使用貨幣
        如需囤積此貨幣需支付其費用

* 貨幣消費比例回饋

		當系統回收貨幣後進行貨幣分發
        分發比例為消費額度/系統淨貨幣資產

* 貨幣穩定性

		貨幣需要穩定性
        其穩定性來自於與法幣的掛鉤

> 依系統排程時間每天結算一次
